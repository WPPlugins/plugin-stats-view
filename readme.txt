== Description ==

The stats of plugin is displayed by shortcode.

== Installation ==

1. Upload `plugin-stats-view` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add a new Page
4. Write a short code. The following text field. `[psview slug="slug-name"]`

How to use

Please add new Page. Please write a short code in the text field of the Page. Please go in Text mode this task.

Examples

[psview slug="plugin-stats-view"]

[psview slug="plugin-stats-view" width="300px" size="small" float="none" icon="60px"]

* slug		Specifies the plugin slug.
* width		Same as CSS of "width". Default: width=NULL
* size		Same as CSS of "font-size". Default: size=NULL
* float		Same as CSS of "float". Default: float="none"
* icon		Icon size. Default: icon="80px"
* border	Same as CSS of "border". Default: border="#CCC 2px solid"
* view		View style. Select "normal" or "simple". Default: view="normal"
* link		You can specify the link destination of the plug-in name. If not specified, this is the plugin homepage link. Default: link=NULL
